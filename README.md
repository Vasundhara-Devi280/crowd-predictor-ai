# Crowd Predictor

## Overview

Crowd Predictor is a real-time crowd monitoring and prediction web application built using the MERN stack.  
The platform allows users to submit live crowd reports for transport locations such as metro stations, bus stops, and railway stations.  
The system visualizes crowd density using analytics, maps, live updates, and prediction logic.

---

## Features

- User Authentication (JWT)
- Register and Login System
- Protected Routes
- Real-Time Crowd Reporting
- Live Dashboard Updates using Socket.IO
- Crowd Analytics Charts
- Crowd Severity Prediction
- Interactive Map Integration
- Search and Filtering
- Responsive UI using Tailwind CSS
- MongoDB Database Integration
